# 🌐 Metlapalli Mehereswar — Portfolio

A premium, hand-crafted personal portfolio website featuring a cinematic loading sequence, 3D canvas animations, dark/light theming, and a fully responsive layout. Built with pure HTML, CSS, and JavaScript — no frameworks, no bloat.

---

## ✨ Features

- **🎬 Cinematic Intro Overlay**: An animated loading screen with a progress bar and initials reveal before the portfolio is shown.
- **🌙 Dark / Light Theme**: Smooth theme toggle with CSS custom properties for full design system consistency.
- **🎨 Custom Design System**: Modular CSS architecture split across `variables`, `base`, `components`, and `sections` for clean maintainability.
- **📄 Resume Download**: Direct access to a downloadable PDF resume.
- **📱 Fully Responsive**: Adapts cleanly to mobile, tablet, and desktop screen sizes.
- **⚡ Zero Dependencies**: Pure vanilla HTML, CSS, and JavaScript — no frameworks, no build step required.

---

## 🛠️ Tech Stack

| Layer | Technology |
|---|---|
| **Markup** | HTML5 |
| **Styling** | Vanilla CSS (Custom Properties, Flexbox, Grid) |
| **Interactivity** | Vanilla JavaScript |

## 🚀 Quickstart

No build step required! Simply open the file in your browser:

```bash
# Clone the repository
git clone https://github.com/mehereswar24/portfolio.git
cd portfolio

# Open directly in your browser
open index.html   # macOS
start index.html  # Windows
```

Or serve it with a simple local server for the best experience:

```bash
npx serve .
# Navigate to http://localhost:3000
```
